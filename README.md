# 🍰 Landing Page - Fernanda Doces

Este foi um projeto real que desenvolvi para a **Fernanda Doces**, com o objetivo de profissionalizar a presença digital dela e facilitar o fechamento de pedidos. 

A ideia aqui foi criar algo prático: o cliente navega pelos doces e, ao decidir, o site já prepara o contato direto pelo WhatsApp, agilizando o atendimento.

### 🚀 O que o projeto faz:
- **Cardápio Digital:** Apresentação clara dos produtos para o cliente.
- **Cupom de Fidelidade:** Uma estratégia que criamos para incentivar o cliente a voltar mais vezes.
- **Integração com WhatsApp:** O foco principal. O botão de pedido já abre a conversa no Whats com as informações necessárias, poupando tempo tanto para a Fernanda quanto para o cliente.

### 🛠️ Tecnologias que usei:
- **HTML5 & CSS3:** Para a estrutura e um visual que combine com a identidade da marca.
- **JavaScript:** Responsável pela lógica de envio das mensagens e interações da página.
